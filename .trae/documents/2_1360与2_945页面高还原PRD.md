## 1. 产品概述
基于提供的 Figma 截图与导出结构，生成五个 1920x1080 的单页信息可视化网页，分别对应“东亚通道”“典型雁种”“迁徙旅程”“结语1”“结语2”，要求高度还原版式、图文层级、素材位置与配色关系。
- 页面用于补充“2025年雁类迁徙信息可视化”专题的通道结构、物种介绍、迁徙机制与结语收束，形成连续的叙事链路。
- 交付目标以桌面端本地预览和截图验收为主，优先保证单屏海报式视觉复现。

## 2. 核心功能
### 2.1 功能模块
1. **东亚通道页面**：展示东亚-澳大利亚迁飞通道、关键节点与典型鸟种分类。
2. **典型雁种页面**：展示五种典型雁类、基础信息、种群结构与趋势。
3. **迁徙旅程页面**：展示迁徙阶段、飞行机制、编队协作与信息共享逻辑。
4. **结语页面组**：展示主题总结、保护结论与返回首页提示。
5. **本地预览入口**：每页均提供可直接打开的静态入口。

## 3. 核心流程
用户打开任一页面后，立即看到完整单屏视觉稿；页面基于固定 1920x1080 画布居中展示，并按照浏览器视口进行等比缩放；用户无需交互即可完成浏览与截图验收。

```mermaid

    A["打开页面"] --> B["加载 1920x1080 固定画布"]

    C --> D["渲染核心图谱、机制或结语结构"]
    D --> E["渲染页脚说明与中英文标题"]
    E --> F["按视口等比缩放展示"]
```

## 4. 用户界面设计
### 4.1 设计风格
- 主色：雾白背景、低饱和粉蓝渐变、棕灰文字
- 辅色：浅蓝用于结构线和种群趋势，浅粉用于环境氛围与视觉平衡
- 标签样式：延续项目统一的细描边矩形标题标签
- 字体建议：中文使用 `PingFang SC`，英文装饰标题保留扩展衬线风格
- 布局风格：海报式绝对定位、留白充足、轻雾化氛围、低对比高级感
- 图形风格：环状图、虚线圆、椭圆底座、飞鸟编队、植物剪影和信息导线
- 通道图风格：底图浅灰、迁飞流线半透明叠加、蓝粉双色路径、节点细描边
- 结语风格：极简留白、双侧渐变雾化背景、轻量文字收束
